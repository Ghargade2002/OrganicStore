# 🌿 OrganicStore - E-Commerce Web Application

A full-stack e-commerce web application built during my internship at ICEICO Technologies Pvt. Ltd.

## 🚀 Features
- 🔐 User Registration & Login
- 🛍️ Product Listing & Detail Page
- 🛒 Add to Cart & Manage Cart
- 📦 Place & View Orders
- ⭐ Product Reviews
- 👤 User Profile Management

## 🛠️ Technologies Used
- **Backend:** PHP, Laravel
- **Frontend:** Blade Templates, HTML, CSS, JavaScript
- **Database:** MySQL
- **Server:** XAMPP
